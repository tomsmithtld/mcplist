# MCP Servers Directory

A curated directory of Model Context Protocol (MCP) servers for Claude Code and Claude Desktop users.

## Features

- 35+ curated MCP servers
- 12 categories
- Easy-to-use interface
- Dark mode support
- One-click install commands

## Deployment

Deploy to Vercel:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_REPO)

## Development

```bash
npm install
npm run dev
```

## Adding Servers

Edit `data/servers.json` to add new servers.
